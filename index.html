The developer's toolkit

What is PHP-Kit ?

PHP-Kit is a set of high quality packages that can be used on any PHP project, irrespective of the framework being used.

Broad in scope, it aims to provide the developer with a set of general purpose libraries that can be easily reused between projects.

The libraries are packaged as Composer packages that can be easily installed into your projects.

We wanted to share these packages with the world (because we believe they can be very useful to any PHP developer) so we grouped them under the same unbrella project, the PHP-Kit.
