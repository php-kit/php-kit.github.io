The developer's toolkit

What is PHP-Kit ?

PHP-Kit is a set of standalone Composer packages that work well together and can be used on any PHP project.

It aims to provide the developer with a set of general/multi purpose libraries and tools that can be easily reused between projects.



PHP-Kit is not a framework


Some of PHP-Kit's packages began life as part of the Selenia framework.
Later on, as they were completely independent from it , we decided to move them here,
thereby allowing any developer to:
  • use them on any kind of PHP project,
  • use them with/without any framework.

Criteria for package acceptance



To be included in PHP-Kit, a package:
  • Must be framework agnostic;
  • Should not depend on packages external to PHP-Kit;
  • Must conform to PSR-4 for autoloading;
  • Must not require a PHP version < 5.4;
  • Adhere to semantic versioning.
    Do not introduce backwards compatibility breaks.
  • Should be well documented;
  • Should be tested;
  • Should be actively maintained;
  • Must be licensed under the MIT, BSD or Apache licenses.

Note: we are aware that, currently, our own packages do not meet all of these criteria.
We are very busy developing the open source Selenia framework and our own commercial projects, so please give us some time to improve these packages.

Of course, if you'd like to help, you're welcome!
